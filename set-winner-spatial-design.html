Aplicación para Apple Vision Pro.

Diseño productos con personas en mente, para que negocios como el tuyo sean confiables.
Vision OS
Conceptual
En abril del año pasado, Apple anunció el lanzamiento de Vision Pro, un innovador dispositivo que fusiona las tecnologías de realidad aumentada y virtual.

¿Por qué crear este producto?

Ofrecía una oportunidad para comprender el proceso de diseño de una interfaz destinada a este dispositivo, así como para seguir las mejores prácticas recomendadas por Apple, con el fin de proporcionar una experiencia de usuario que sea tanto utilizable como atractiva.

¿Cuál fue el objetivo?

En términos generales, el objetivo es proporcionar una experiencia inmersiva y satisfactoria para el usuario, sin comprometer la usabilidad ni causar fatiga durante el uso del producto.
